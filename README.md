FileShare
=========

Minimal file sharing service written in Rust

Quick start
-----------

- Build locally: `cargo build --release`
- Run locally: `cargo run --release` or `./target/release/fileshare`
- Docker (recommended): `docker compose up --build`

Notes
-----

- The project includes a small web UI in the `templates/` folder.
- See `src/` for the server implementation and configuration options.
